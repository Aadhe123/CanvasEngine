# Canvas Engine
http://canvasengine.net

## Description

Framework to create video games in HTML5 Canvas

## Features

* Scene Structure
* Multiplayer model
* Preloading
* Animations
* Timeline
* Keyboard and Gamepad
* Sound
* Scrolling
* Level Design
* Save & Load
* Spritesheet Management
* Mouse Control
* Elements Manipulation

## Roadmap

* Windows
* Transition
* Overlay scenes
* Game Effects
* Text Effects
* Synchronized multi canvas (local multiplayer)
